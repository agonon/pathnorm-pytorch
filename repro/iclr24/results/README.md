# ICLR 2024 Results

This folder contains the ICLR 2024 results included with the repository.

- `paper_release/` is the default notebook source.
- `rerun/` is for local reruns and is ignored by git.

Included results:

- `paper_release/1_compute_pretrained_pathnorm_margins_resnets` stores the pretrained path-norm and margin results.
- `paper_release/2_train_imp` and `paper_release/3_plot_imp` store the included Figure 4 rerun (`40` epochs, `8` pruning iterations).
- `paper_release/4_train_increasing_dataset` and `paper_release/5_plot_increasing_dataset` store the included Figure 5 rerun (`1` seed).
