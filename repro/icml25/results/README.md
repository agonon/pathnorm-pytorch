# ICML 2025 Results

This folder contains the ICML 2025 results included with the repository.

- `paper_release/` is the default notebook source and contains the included one-seed pruning rerun.
- `rerun/` is the staging location for local reruns and is kept empty in the repository.

Included files:

- `benchmark_summary.csv`
- `dense_summary.csv`
- `pruning_accuracy_body.csv`
- `training_curves_0.4.json`
- `plot_training_curves_test0.4.pdf`
- `plot_training_curves_test0.4.png`
- `pruning_accuracy_full.csv`
